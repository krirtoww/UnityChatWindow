如果不用SVG示例里的图标可能无法正常加载，自己替换下。

Plugins to install
Localization
DOTween
(Optional) Load SVG: com.unity.vectorgraphics
If you don't use the icons in the SVG example, they may not load properly. Please replace them yourself.
In addition, if you are using version 2022 or earlier, there may be some issues with the TMP and Localization components. Please fix them yourself.
There should be no issues with version 2023.
